# RAP III

Простой генератор рифм для фристайла на русском языке, написанный на Python.

---

# Установка

Установка зависимостей (требуется установленный Python 3.6+ и Google Chrome):
```shell
python -m pip install ./requirements.txt --user
```

# Запуск

Запуск программы:
```shell
python ./generator.py
```
